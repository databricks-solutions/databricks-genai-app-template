Databricks does not offer official support for Databricks Solutions and its repository.
For any issue with this assets or the demos installed, please open an issue using github and the team will have a look on a best effort basis.

## Data collection
To improve users experience and asset quality, this template sends anonymized report usage. This information is captured for product improvement only and not for marketing purpose. By using this repo and the assets it provides, you consent to this data collection. If you wish to disable it, edit the app.json file
